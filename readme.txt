Безопасный перенос терминов и связей товаров между таксономиями WooCommerce.

== Description ==

Плагин добавляет инструмент в админку WooCommerce для переноса терминов и связей товаров между таксономиями. Поддерживает предварительный просмотр, пакетную обработку, остановку процесса, логирование и очистку пустых WooCommerce-атрибутов.

== Installation ==

1. Загрузите плагин через админку WordPress или установите ZIP из WCS update server.
2. Активируйте плагин.
3. Откройте WooCommerce -> Taxonomy Migrator или ссылку Settings в списке плагинов.
